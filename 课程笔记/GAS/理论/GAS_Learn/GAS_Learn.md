# 组成部分
#名词解释 GAS (Gameplay ability system)分为七个部分
	1：Gameplay ability （游戏玩法）
	2：Attributes and Attribute Sets（属性与属性集）
	3：[[Gameplay Effect]]（游戏效果）
	4：Gameplay Cue（游戏提示）
	5：Ability System Component（核心）
	6：Gameplay Tag
	7：Ability Task
